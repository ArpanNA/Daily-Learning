## Question
You are tasked with implementing a function that determines whether a given integer array, `nums`, can be transformed into a non-decreasing array by altering the value of *at most one* element. A non-decreasing array satisfies the condition `nums[i] <= nums[i+1]` for all valid indices `i`.

Your implementation must handle the case where the violation occurs at the very beginning of the array (`i=0`).

Function Signature:
python
def check_non_decreasing(nums: list[int]) -> bool:
    pass


Example 1:
Input: nums = [4, 2, 3]
Output: True 

Example 2:
Input: nums = [4, 2, 1]
Output: False

## My Answer
```text
def check_non_decreasing(nums: list[int]) -> bool:
    changed = False   # we are allowed to change at most one element

    for i in range(len(nums) - 1):
        if nums[i] > nums[i + 1]:     # found a violation

            if changed:
                return False          # second violation â†’ cannot fix with one change

            # Decide how to fix:
            # If this is the first element, or lowering nums[i] won't break the previous order
            if i == 0 or nums[i - 1] <= nums[i + 1]:
                nums[i] = nums[i + 1]     # pretend we lower nums[i]
            else:
                nums[i + 1] = nums[i]     # otherwise, raise nums[i+1]

            changed = True

    return True

```
