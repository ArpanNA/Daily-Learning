## Question
In the context of modern operating systems, if both represent units of execution, what fundamental resource is owned individually by a process but is typically shared among all threads belonging to the same process?

This answer is precisely correct. The Process's Address Space (Virtual Memory) is the fundamental resource owned by the process, containing shared segments like the code, heap, and global data, which all threads within that process access.
